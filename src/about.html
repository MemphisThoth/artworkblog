About

I have been painting with oil paints and watercolors for over nine years now. I wanted to create a blog website to highlight the knowledge I have gained over the years.

This website was created using 11ty. It is a static website generator using javascript as an enigine. I create posts using markdown files, and 11ty applies liquid templates to structure and style those files.
